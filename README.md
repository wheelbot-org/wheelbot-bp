# Чертежи и 3D модели проекта wheelbot

В папке `fcstd` лежат чертежи деталей в формате FreeCAD. Каждый файл - отдельная деталь. В папке `3mf` лежат уже подготовленные для 3D печати модели. Имена файлов в папках совпадают, отличаются только разрешения.

Все чертежи адаптированы под использование подшипников от роликовых коньков - стандарт 608 (8x22x7 mm), внутренний диаметр 8 mm. Выбор обусловлен их распрастраненностью и ценой. Установка подшипников в детали предполагается запресовкой (с натягом).

Для разъемных соединений используются винты M4 цилиндрический с внутренним шестигранником и гайки к ним. Выбор обусловлен их распрастраненностью, ценой, потребностью многократной сборки и разборки, скрытие шляпки.

Список не печатных деталей по [ссылке](./EXTERNAL-PARTS.md).

## Базовые узлы

### Рама

Рама с ребрами жесткости (изгиб, кручение) - 100x200 mm, для того, чтобы деталь помещалась на столы базовых 3D принтереров. На раме сеткой 25x25 mm размещены отверстия под винты с пазами для скрытия шляпки, на обратной стороне пазы для запресовки гаек.

|  Код | Название | Описание | Файл | Превью |
|---|-------|----------|--------------|---|
